📋 Setup Guide (5 minutes, free)
  1. Go to console.firebase.google.com and sign in with your Google account.
  2. Click Add project → name it anything → click through and create.
  3. In the left sidebar click Build → Realtime Database.
  4. Click Create database → choose any location → select "Start in test mode" → Enable.
  5. Copy the Database URL shown at the top (looks like https://your-project-rtdb.firebaseio.com).
  6. Paste it in the field above → click Save & Test Connection.
  7. Click ⬇ Download config.js → upload that file to your Netlify site (drag & drop into Netlify deploy).
